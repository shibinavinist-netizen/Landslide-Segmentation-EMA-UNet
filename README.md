# Landslide Segmentation using EMA-UNet

## Overview

This project implements a deep learning-based landslide segmentation framework using Efficient Multi-Scale Attention (EMA) integrated with U-Net architecture.

## Features

* Pixel-level landslide segmentation
* Efficient Multi-Scale Attention (EMA)
* Deep Learning based semantic segmentation
* Satellite image analysis
* High boundary precision

## Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* OpenCV
* Rasterio
* GDAL

## Dataset

* Sentinel-2 Satellite Imagery
* ALOS PALSAR Data
* Landslide4Sense Dataset

## Results

* Accuracy: 97.04%
* Precision: 69.71%
* Recall: 63.62%
* F1 Score: 66.52%

## Future Improvements

* Real-time landslide monitoring
* UAV deployment
* Multi-hazard detection
